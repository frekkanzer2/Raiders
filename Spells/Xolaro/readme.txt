// Xolaro has 6 PA
Xelor Dial - Summons a Xelor Dial
Temporal Paradox - At the start of the turn, the target will switch position with the dial.
	At the end of the turn, the target will re-switch position with the dial.
	If the dial is not summoned, switch position with a random ally.
Overclock - If the caster is adjacent to the dial, immediately gains 2 PA.
	40% chance to remove 2 PA from the target.
Petrification - If the target is adjacent to the dial, remove 4 PA.
	In the next turn, it will gain 4 PA.
